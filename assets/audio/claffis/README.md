# Claffis practice note placeholders

These are synthetic placeholder tones used by Claffis Practice Mode.

Files are mapped by hotspot ID:

- T1.wav, T2.wav, T3.wav
- M1.wav, M2.wav, M3.wav, M4.wav, M5.wav
- B1.wav, B2.wav, B3.wav, B4.wav

You can replace them with your own local samples later. Keep the same filenames.
