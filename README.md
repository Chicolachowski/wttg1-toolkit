# WTTG 1 Toolkit

A single-file multi-page helper hub for Welcome to the Game 1.

## Included subpages

- Dashboard
- Claffis Helper
- Key Compiler
- A.N.N Notes Tracker
- Session Checklist

## How to run

Open `index.html` in your browser.

## Notes

Everything is stored in `localStorage`, so your notes and selections stay saved in the same browser.
The app uses hash routes like `#claffis`, `#keys`, `#links`, and `#checklist`, so it can be moved to React Router later without changing the concept.
